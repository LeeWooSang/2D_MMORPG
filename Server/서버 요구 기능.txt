[플레이어 관련]
1. 접속 중인 플레이어 수
2. 플레이어 목록 (ID, 닉네임, 위치, 상태 등)
3. 개별 플레이어 정보 (HP, 경험치, 장비, 버프/디버프 등)
4. 플레이어 이동 로그/히스토리
5. 플레이어 핑/지연 시간

[월드/맵 정보]
1. 월드 별 플레이어 분포
2. 맵 상태 (로드 상태, 몬스터 수, 이벤트 진행 여부 등)
3. 스폰 포인트/던전 현황

[서버 상태]
1. CPU/메모리 사용량
2. GC 상태
3. 네트워크 패킷 처리량 (in/out)
4. TPS (tick per second) / 프레임
5. DB 연결 상태 / 쿼리 속도

[네트워크 및 통신]
1. 연결된 클라이언트 목록 (IP, 포트, 세션 시간)
2. 실시간 패킷 모니터링 (필터링 가능)
3. 문제가 발생한 패킷 로그
4. 패킷 처리 큐 상태

[디버깅/운영 기능]
1. 실시간 로그 뷰어 (레벨별 필터: DEBUG/INFO/WARN/ERROR)
2. 특정 플레이어 강제 추방 or 텔레포트
3. 서버 명령어 실행창 (예: /kick, /broadcast 등)
4. 실시간 설정 변경 (ex: 이벤트 On/Off)
